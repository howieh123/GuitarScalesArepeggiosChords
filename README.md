## Run Locally

**Prerequisites:**  Node.js


1. Clone the Repo
   `git clone https://github.com/howieh123/GuitarScalesArepeggiosChords.git` 
2. Install dependencies:
   `npm install`
3. Run the app:
   `npm run dev`
